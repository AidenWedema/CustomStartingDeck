Adds a custom run modifier that let's you pick any card to put into your deck. Change the size of your deck in the mod configuration.

Credits to Pikcube in the for making the scripts used to add new custom run modifiers. Check out [Pikcube.Common](https://github.com/pikcube/Pikcube.Common)
